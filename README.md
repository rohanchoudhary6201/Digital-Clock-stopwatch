# Digital Clock & stopwatch
A modern and interactive Digital Clock & Stopwatch web application built using HTML, CSS, and JavaScript.

🚀 Live Demo

Visit the project:
https://rohanchoudhary6201.github.io/Digital-Clock-stopwatch/

📌 Features
Real-time Digital Clock
Accurate Stopwatch with Millisecond Precision
Start, Pause, Reset, and Lap Functions
Modern 3D Glassmorphism Design
Animated Background Effects
Responsive Layout for Desktop and Mobile
Attractive Neon UI with Premium Color Theme
Built using Pure HTML, CSS, and JavaScript
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
